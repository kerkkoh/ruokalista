Simple API for fetching some relevant information from this site.

SIMPLE QUERY

  • Query endpoint
    ruokalista.glitch.me (https)
  • Method
    GET
  • URL
    /api
  • URL Params

    1. id - (INTEGER) - This is the number which gets incremented to the current day, eq. id=0 would be the present day and id=-1 would be the day before the present day.
    2. q - (STRING) - This is either koulut or pvkoti
  • Note: This is not accurate with holidays and hence the use should be avoided during the time.
  • Sample query:
    https://ruokalista.glitch.me/api?id=0&q=koulut
    or
    $.ajax({
  url: "/users/1",
  dataType: "json",
  type : "GET",
  success : function(r) {
    console.log(r);
  }
});
  • Sample Response:

    {"food":"Lohikiusaus / Kasvispyörykkä","day":5,"dayStr":"Perjantai","week":3}
